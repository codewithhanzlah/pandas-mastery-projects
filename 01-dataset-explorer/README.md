# 📂 Dataset Explorer Tool (Pandas Project 1)

This project is part of the **Pandas Mastery Project Series**, focusing on basic data loading and exploration.

---

## 🚀 Overview

The Dataset Explorer Tool provides a quick way to understand the structure and quality of a dataset. It helps in inspecting data before performing any analysis or preprocessing.

---

## 📊 Dataset

**Iris Dataset (Kaggle)**
https://www.kaggle.com/datasets/uciml/iris

* File used: `Iris.csv`

---

## 🔍 Features

* View first and last rows (`head`, `tail`)
* Check dataset shape
* Inspect column names
* Get dataset information (`info`)
* Detect missing values
