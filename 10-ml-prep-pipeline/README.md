# 🤖 End-to-End ML Prep Pipeline (Pandas Project 10)

This project is part of the **Pandas Mastery Project Series**, focusing on building a complete data preprocessing pipeline for machine learning.

---

## 🚀 Overview

This project implements an end-to-end data preprocessing pipeline using the Titanic dataset. It integrates data cleaning, feature engineering, and encoding techniques to transform raw data into a structured, machine-learning-ready dataset.

---

## 📊 Dataset

**Titanic Dataset (Kaggle)**
https://www.kaggle.com/datasets/whenamancodes/titanic-dataset-machine-learning-from-disaster

* File used: `train.csv`

---

## 🔍 Features

* Handling missing values using appropriate strategies
* Dropping irrelevant or redundant columns
* Creating new features (e.g., family size, isolation indicator)
* Encoding categorical variables using one-hot encoding
* Applying structured data transformation steps
* Validating dataset consistency after preprocessing
* Exporting final machine-learning-ready dataset
