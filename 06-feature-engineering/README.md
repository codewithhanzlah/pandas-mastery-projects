# 🛠️ Feature Engineering Toolkit (Pandas Project 6)

This project is part of the **Pandas Mastery Project Series**, focusing on creating and transforming features for machine learning workflows.

---

## 🚀 Overview

The Feature Engineering Toolkit enhances raw data by creating meaningful and informative features that improve the performance of machine learning models. It demonstrates how to transform existing variables into more useful representations.

---

## 📊 Dataset

**Titanic Dataset (Seaborn / GitHub Source)**
https://raw.githubusercontent.com/mwaskom/seaborn-data/master/titanic.csv

* File used: `titanic.csv`

---

## 🔍 Features

* Creating new features (e.g., family size, derived metrics)
* Generating binary features for classification tasks
* Binning continuous variables into categorical groups
* Applying lambda functions for custom transformations
* Encoding categorical variables for modeling
* Exporting feature-enhanced dataset
