# 🎓 Student Performance Analyzer (Pandas Project 8)

This project is part of the **Pandas Mastery Project Series**, focusing on analyzing structured data using aggregation and conditional logic.

---

## 🚀 Overview

The Student Performance Analyzer evaluates academic performance across different classes and subjects. It demonstrates how to analyze educational data to identify trends, top performers, and overall performance metrics.

---

## 📊 Dataset

**Synthetic Student Dataset (Generated using NumPy)**

* Created within the notebook for controlled analysis
* Includes:

  * Student names
  * Class groups (A, B, C)
  * Subjects (Math, Science, English)
  * Marks

---

## 🔍 Features

* Calculating average marks by subject and class
* Identifying overall and subject-wise top performers
* Applying pass/fail classification using conditional logic
* Computing pass percentage across different classes
* Extracting insights from grouped academic data
* Exporting processed dataset
