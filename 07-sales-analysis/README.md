# 📊 Sales Data Analyzer (Pandas Project 7)

This project is part of the **Pandas Mastery Project Series**, focusing on data analysis using GroupBy operations and aggregation techniques.

---

## 🚀 Overview

This project analyzes sales data to extract meaningful business insights across different cities and product categories. It demonstrates how to use Pandas for aggregating data and identifying performance trends.

---

## 📊 Dataset

**Synthetic Sales Dataset (Generated using NumPy)**

* Created within the notebook for controlled analysis
* Includes:

  * City
  * Product
  * Sales values

---

## 🔍 Features

* Grouping data by city and product categories
* Aggregating metrics (total sales, average sales)
* Performing multi-level grouping for deeper analysis
* Identifying top-performing and low-performing segments
* Extracting insights from structured datasets
