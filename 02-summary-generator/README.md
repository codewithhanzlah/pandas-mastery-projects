# 📊 Dataset Summary Generator (Pandas Project 2)

This project is part of the **Pandas Mastery Project Series**, focusing on generating statistical summaries and insights from datasets.

---

## 🚀 Overview

The Dataset Summary Generator provides a quick statistical overview of a dataset. It helps in understanding data distribution, identifying patterns, and preparing for deeper analysis.

---

## 📊 Dataset

**Titanic Dataset (Kaggle)**
https://www.kaggle.com/datasets/yasserh/titanic-dataset

* File used: `Titanic-Dataset.csv`

---

## 🔍 Features

* Summary statistics (`mean`, `median`, `std`)
* Unique values per column
* Value counts for categorical data
* Detection of data distribution patterns
* Clean and structured summary output
