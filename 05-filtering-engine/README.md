# 🔍 Data Filtering Engine (Pandas Project 5)

This project is part of the **Pandas Mastery Project Series**, focusing on advanced data filtering and querying techniques using Pandas.

---

## 🚀 Overview

The Data Filtering Engine demonstrates how to extract relevant subsets of data using flexible and efficient filtering methods. These techniques are essential for data analysis, feature selection, and decision-making workflows.

---

## 📊 Dataset

**Titanic Dataset (Seaborn / GitHub Source)**
https://raw.githubusercontent.com/mwaskom/seaborn-data/master/titanic.csv

* File used: `titanic.csv`

---

## 🔍 Features

* Single-condition filtering for basic data selection
* Multi-condition filtering using logical operators
* Boolean masking for advanced filtering
* Query-based filtering using `.query()`
* Filtering using `isin()` for multiple values
* Selecting specific columns for analysis
* Exporting filtered datasets
