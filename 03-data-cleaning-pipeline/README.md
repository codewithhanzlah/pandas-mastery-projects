# 🧹 Data Cleaning Pipeline (Pandas Project 3)

This project is part of the **Pandas Mastery Project Series**, focusing on cleaning and preprocessing real-world datasets.

---

## 🚀 Overview

This project implements a data cleaning pipeline that transforms raw Titanic dataset into a clean and analysis-ready format. It demonstrates essential preprocessing steps required before performing data analysis or building machine learning models.

---

## 📊 Dataset

**Titanic Dataset (Seaborn / GitHub Source)**
https://raw.githubusercontent.com/mwaskom/seaborn-data/master/titanic.csv

* File used: `titanic.csv`

---

## 🔍 Features

* Handling missing values using statistical methods (median, mode)
* Removing duplicate records
* Standardizing column names for consistency
* Correcting data types
* Exporting cleaned dataset for downstream tasks
