# 🔤 Categorical Data Processor (Pandas Project 4)

This project is part of the **Pandas Mastery Project Series**, focusing on preprocessing and encoding categorical data for machine learning workflows.

---

## 🚀 Overview

This project processes categorical features by cleaning, standardizing, and transforming them into numerical formats suitable for machine learning models. It demonstrates essential techniques for handling non-numeric data in real-world datasets.

---

## 📊 Dataset

**Titanic Dataset (Seaborn / GitHub Source)**
https://raw.githubusercontent.com/mwaskom/seaborn-data/master/titanic.csv

* File used: `titanic.csv`

---

## 🔍 Features

* Standardizing categorical values (lowercasing, trimming whitespace)
* Handling inconsistent text entries
* Binary encoding for two-category variables
* One-hot encoding for multi-category features
* Exporting processed dataset for downstream use
